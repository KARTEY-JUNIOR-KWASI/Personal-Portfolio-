# BigJuu Tech Portfolio

Modern single-page portfolio inspired by animated Dribbble shots. Built with semantic HTML, modern CSS, and expressive vanilla JavaScript interactions.

## Getting Started
- Open `index.html` in your browser to explore the site.
- Edit `styles.css` and `app.js` to adjust the theme, animations, and motion behaviors.

## Key Features
- Hero section with typed focus states, animated blobs, and glassmorphism card.
- Projects grid with hover tilt, reveal animations, and accessible modal details.
- Mobile-aware navigation, smooth scrolling, and client-side README download utility.
